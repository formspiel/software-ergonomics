Character Size Checker Draft

A bookmarklet for in-browser character-height accessibility checks

Run this script on any web page to check whether text meets the minimum character height requirements of ISO 9241-303. Elements are highlighted green (pass) or red (fail) based on their computed font size and family. A control panel lets you toggle individual checks on and off.

What it checks

  • Character height per font family against a 10 px minimum
  • Missing or empty alt attributes on images
  • lang attribute presence on elements

Installing as a bookmarklet

  1. Open the script (link below) and copy its entire contents.
  2. Create a new bookmark in your browser — any page, any name.
  3. Edit the bookmark and paste the script into the URL / Address field, replacing the existing URL.
  4. Save. The bookmark now runs the script when clicked on any page.

Show your bookmarks bar first if it is hidden: Chrome / Edge Ctrl+Shift+B · Firefox Ctrl+Shift+B · Safari ⌘+Shift+B
