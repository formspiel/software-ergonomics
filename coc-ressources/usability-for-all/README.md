# Usablility for All
This repository is the *playground* of @Andreas_Helge ([Centre of Competence for Ergonomics & Usability](https://www.amh.me)) to understand how universal design for the web works and *share* the knowledge with the developers to make our websites and (mostly) web applications more usable and accessible for all of our clients and employees. And to fulfil the requirements for software development of Allianz Group.
