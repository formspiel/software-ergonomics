Invisible Skip Links

Skip links inspired by Einfach für Alle plus interactivity. On Safari (Mac) use alt-tab or enabled tab option

The sectret that this technique works is the tabindex="-1" attribute for the relevant sections.

For screen reader output of the interactivity you need the aria-live attribute to trigger the screen reader that it reads the text changes immediatly.
