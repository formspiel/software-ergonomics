Cap-Height Ratio Verification

Display Character Size Calculator · ISO 9241-303 · One sheet per device

Cap-height ratios in the calculator are published per-em estimates. This sheet guides you through measuring the actual rendered cap height on a specific device and browser, so the estimates can be verified or corrected.

Tool URL: http://localhost:8080/tool-display-legibility/  (run python3 -m http.server 8080 from the repo root, or open the deployed URL)

Equipment

  • The device you are testing (any screen)
  • A browser on that device (note which one)
  • An image editor for measuring pixels: Preview (macOS) or Paint / Paint 3D (Windows)

Step-by-step instructions

  1. Open the tool on the device you are testing.
  2. Select the device from the Preset list, or enter its specifications manually (width × height px, diagonal, DPR).
  3. Set the Viewing distance appropriate for the device type.
  4. Click Calculate.
  5. Scroll to Font compliance check. Select the font you want to verify and set the size to 16 CSS px.
  6. Set browser zoom to exactly 100 % — use ⌘0 (macOS) or Ctrl0 (Windows).
  7. Take a screenshot of the render area (the white box showing the test string):
    • macOS: ⌘⇧4 → drag to select the render box → captures at dpx
    • Windows: ⊞⇧S → drag to select → captures at CSS px
  8. Open the screenshot in your image editor. Measure the height of the capital E (first character) in pixels — from the top of the letter to the baseline.
  9. Enter this pixel value in the Measure & verify section of the tool. Select the correct unit (CSS px or dpx).
  10. Note the Measured cap ratio shown. Record it in the table below.
  11. Repeat from step 5 for each additional font.
Tip — measuring in Preview (macOS): Open screenshot → Tools → Adjust Size to confirm dimensions, or use the selection tool and note the H value in the info bar. In Pixelmator or Photoshop, draw a selection from cap-top to baseline; read the height in the info panel.

Tip — measuring in Paint (Windows): Open screenshot → use the selection tool → drag from cap-top to baseline. The status bar shows "W × H px". Note the H value.
