Tab close prevention

Enter something in the fields below, then try to close the tab or window. If unsaved data is present, the browser will ask you to confirm before leaving — protecting the user from accidentally losing their input.

How it works

The warning dialog is triggered by the browser's beforeunload event. One important limitation: the dialog text and appearance cannot be customised by the application. Browsers intentionally blocked this years ago to prevent phishing sites from mimicking system prompts. What you see is always the browser's own native dialog, regardless of which browser or operating system the user is on.
